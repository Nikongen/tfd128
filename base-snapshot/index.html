Der TFD 128

Mit dem “Temperatur-Feuchte-Datenlogger 128” hat die Firma ELV ein günstiges Produkt zur elektronischen Überwachung von Temperatur und Feuchtigkeit entwickelt (das Gerät wird u.a. auch von der Firma Conrad Electronic vertrieben). Das Gerät speichert dabei regelmäßig und - im Batteriebetrieb - unabhängig vom Computer die entsprechenden Umweltdaten. Anschließend können diese dann am PC abgerufen und weiterverarbeitet werden.

Die dem Gerät beiliegende Software baut auf Java-Technologie auf, steht aber nur für Windows zur Verfügung. Mit Hilfe des hier verfügbaren Python-Skriptes ist es möglich, den TFD 128 unter Linux und Mac OS X von der Kommandozeile aus anzusprechen.

Hinweis: Bei Anschluss an den USB-Port sind zum Betrieb keine Batterien notwendig, der TFD128 bezieht dann die notwendige Energie direkt vom Rechner.

Wenn Sie auf der Suche nach einer alternativen Software für Windows sind, sehen Sie mal auf der Seite von Vincenz Dreger nach.

The TFD 128

ELV Elektronik created a cheap device to electronically log room-temperature and humidity, the TFD 128 (the device is also being sold by Conrad Electronic). In predefined intervals and - if running on batteries - independently from the PC, the small device collects the environmental data. Afterwards, this data can be downloaded to the PC and further processed.

The software provided with the device is build upon Java technology, but only supports Windows systems. If you want to control the TFD 128 with Linux or Mac OS X, you can find a python-script in the download section to access the data via command line.

Remark: If the TFD 128 is connected to the USB-port, no batteries are required since the device will be powered by the PC.

If you’re looking for an alternative Windows software, visit Vincenz Dreger’s page.
